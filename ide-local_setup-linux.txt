I.	APACHE SERVER

	// restart the apache server

	systemctl restart apache2

	sudo /etc/init.d/apache2 restart

	touch index.html

II.	PHP

	sudo apt-get install php	// installs latest stable version	

To install PHP 7.3 you�ll need to use a third-party repository.

Switch between PHP versions:
